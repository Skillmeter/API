Candidate
====

A candidate represent a person assigned to take one or more tests on Skillmeter. 

* [Creating a Candidate](#creating-a-candidate)
* [Fetch a Candidate](#fetch-a-candidate)
* [Deleting a Candidate](#deleting-a-candidate)


### Creating a Candidate

  POST /candidate
  
#### Input

```json
{
"First_Name":"John",
"Last_Name":"Doe",
"Email": "john-doe@domain.com",
"Language" : "en",
"ExternalId" : "123",  
"Tests": [30, 33],
"AllowResume": false,
"Feedback": false
}
```

#### Response

  Status: 201 Created


### Fetch a Candidate

  GET /candidate/{id}
  
  GET /candidate/{externalId}
  
#### Response

### Deleting a Candidate

DELETE /candidate/:id

#### Response

  Status: 204 No Content
